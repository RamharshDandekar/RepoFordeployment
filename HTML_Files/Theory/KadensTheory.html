Kadane's Algorithm

Introduction to Kadane's Algorithm

Kadane's Algorithm is an efficient method used to find the maximum sum of a contiguous subarray within a one-dimensional array of numbers. It operates in linear time, making it optimal for large datasets.

Problem Statement

Given an integer array, the goal is to find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

How Kadane's Algorithm Works

  • Initialize two variables: max_so_far and max_ending_here. Set both to the first element of the array.
  • Iterate through the array starting from the second element. For each element, update max_ending_here to be the maximum of the current element or the sum of max_ending_here and the current element.
  • Update max_so_far to be the maximum of itself and max_ending_here.
  • Continue this process until the end of the array. The value of max_so_far will be the maximum sum of the contiguous subarray.

Example

For the array [−2,1,−3,4,−1,2,1,−5,4], Kadane's Algorithm will find that the maximum sum subarray is [4,−1,2,1], which sums to 6.

Complexity Analysis

The time complexity of Kadane's Algorithm is O(n), where n is the number of elements in the array. The space complexity is O(1), as it uses a constant amount of space.

Applications of Kadane's Algorithm

  • Finding maximum profit in stock trading, where daily price changes can be represented as an array.
  • Image processing to identify the maximum pixel intensity region.
  • Dynamic programming problems where the maximum sum subarray needs to be determined.

Variations of Kadane's Algorithm

  • Modified Kadane’s Algorithm can be used to find the maximum sum subarray with at most one negative number.
  • Two-dimensional Kadane’s Algorithm can find the maximum sum rectangle in a 2D array.
